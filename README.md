# Superstore Sales Dashboard

## Overview
This project analyzes Superstore Sales data using Microsoft Excel. Pivot Tables, Charts, and Conditional Formatting were used to create an interactive dashboard.

## Features
- Total Sales Analysis
- Profit by Category
- Sales Trends Over Time
- Pivot Tables & Charts
- Conditional Formatting for High/Low Performers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting

## Files
- `Superstore_Sales_Dashboard.xlsx` – Dashboard and analysis workbook
